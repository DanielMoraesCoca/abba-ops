# Jornada do Cliente — mapa do funil

> Um doc por estágio, formato fixo: **Dono · Entrada · Checklist · Saída · Ferramentas · Templates · Prazo-alvo**. Cada doc cabe numa página e é executável por uma pessoa numa sentada. Quando um cliente muda de estágio, abra o doc do estágio novo e siga o checklist — é isso.

## O funil

```
  AQUISIÇÃO                    VENDA                      ENTREGA                          RELAÇÃO
┌─────────────────┐   ┌──────────────────┐   ┌──────────────────────────────────┐   ┌────────────────────┐
│ 01 Visitante    │   │ 03 Proposta      │   │ 05 Onboarding                    │   │ 10 Alinhamento     │
│    e lead       │ → │ 04 Contrato      │ → │ 06 Avaliação profunda            │ → │    com o conselho  │
│ 02 Degustação   │   │                  │   │ 07 Construção e implantação      │   │ 11 Renovação e     │
│    gratuita     │   │                  │   │ 08 Capacitação e transformação   │   │    encerramento    │
└─────────────────┘   └──────────────────┘   │ 09 Manutenção                    │   └────────────────────┘
                                             └──────────────────────────────────┘
                          (o estágio 10 acontece DURANTE a entrega também — é a constante)
```

## Métricas de funil (revisão mensal na reunião de sócios)

| Transição | Métrica |
|---|---|
| Visitante → lead | leads/mês por origem |
| Lead → degustação | % que aceita a análise gratuita |
| Degustação → proposta | % convertida |
| Proposta → contrato | % fechada · ciclo (dias) · ticket médio |
| Entrada → programa | % de produtos de entrada que viram programa em 90 dias |
| Programa → manutenção | % que assina continuidade |
| Cliente → indicação | indicações por encerramento |
